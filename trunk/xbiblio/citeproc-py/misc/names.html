How names should be written to be properly detected

  • Western names:
    1. John Doe
    2. J. Doe | J.T. Doe | J. T. Doe
      Use points to indicate initials.
    3. John T. Doe
      The initial for the middle name needs to be succeeded with a point.
    4. T. John Doe
      The initial for the first given name needs to be succeeded with a point.
    5. Joh. S. Bach
      Initals can consist of more than one letter for some names. Some styles may still choose to abbreviate these to a single letter anyway.
    6. J.W. (Hans) Bakker
      Use brackets for people for whom the first letter(s) of their given name differs from their initials.
    7. Klaas de Vries
      Use lowercase for articulars. Articulars are displayed before a family name, but are ignored during sorting.
    8. John Doe Sr.
      Use a point after a suffix.
    9. John Doe IV
      Roman numbers are automatically detected as suffixes.
    10. Pope (John Paul) II
      Put brackets around a given name if no family name is available.
    11. (Jaap) Sinninghe Damsté
      Put brackets around a given name if the family name contains a space.
  • Eastern names:
    1. Mao Zedong
    2. Setiadi
      Some Asian people go by a single name. Those won't be initialized in any style.
  • Icelandic names:
    1. Björk Guðmundsdóttir
      Icelandic names will be sorted by their given name.
